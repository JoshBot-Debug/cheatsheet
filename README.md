# Cheatsheet for web development
This repo should be treated as a quick review of basic web developer knowledge.

## Who is this?
This is for both **backend** and **frontend** developers. Every web developer must be knowledgable on theses subject.

This is not for **beginners** nor for **intermediate** nor **advanced** level engineers. It's for everyone. Everything listed is not limited to any specific level of skill. We perfer to cover everything beginner, intermediate & advanced depth and topics.

**The whole truth, not just "Enough for a fresher"**

## Why is this?
